# Análise de Acidentes de Trânsito com Python e MySQL

Projeto de análise de dados utilizando dados públicos de acidentes de trânsito.
O objetivo é explorar padrões de ocorrências, armazenar dados tratados em banco
de dados relacional e responder perguntas através de consultas SQL.

## Tecnologias
- Python
- Pandas
- MySQL
- SQL
- Jupyter Notebook

## Etapas
- Análise exploratória
- Modelagem de banco de dados
- Inserção de dados no MySQL
- Consultas SQL para geração de insights

## Perguntas respondidas
- Qual o horário com mais acidentes?
- Qual o dia da semana com mais ocorrências?
- Quais bairros concentram mais acidentes?
